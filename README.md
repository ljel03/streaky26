\# Streaky



Streaky je jednoduchá mobilní aplikace pro sledování streaků (opakovaných návyků).

Aplikace je vytvořena pomocí Ionic + Angular (standalone components) a běží na platformě Android pomocí Capacitoru.



---



\## Funkcionalita



\- Okna:

  - hlavní obrazovka (Home)

  - obrazovka pro přidání streaku (Nový streak)



\- Zobrazení aktuálního dne v hlavičce aplikace

&nbsp; - přepínání dne do budoucna pomocí šipek



\- Správa streaků:

&nbsp; - přidání nového streaku

&nbsp; - zobrazení:

&nbsp;   - aktuálního streaku

&nbsp;   - maximálního dosaženého streaku

&nbsp; - reset streaku

&nbsp; - smazání streaku



\- Perzistence dat:

&nbsp; - streaky jsou ukládány lokálně do zařízení pomocí `@capacitor/preferences`

&nbsp; - data zůstávají zachována i po restartu aplikace



\- REST API:

&nbsp; - aplikace načítá náhodný citát z veřejného REST API

&nbsp; - možnost ruční obnovy citátu
